HTML: HyperText Markup Language

  • Language that web browsers use to render pages
  • First released in 1993
  • Designed to create web documents. Has since been extended for web applications

Cascading Style Sheets

  • Language for describing the presentation of a document
  • First released in 1996
  • Evolved alongside HTML
  • Most browsers support CSS 3

Exercise 5 - On board element click

Modify function onBoardPieceClick to add a class to clicked board piece (available in event.target). The class name should be equal to value of variable turn created in Excercise 1.

Exercise 5 - Solution


    function onBoardPieceClick(event) {
        var boardPiece = event.target
        boardPiece.classList.add(turn)
    }
        
    for (var i = 0; i < boardPieces.length; i = i + 1) {
        boardPieces.item(i).onclick = onBoardPieceClick;
    }
