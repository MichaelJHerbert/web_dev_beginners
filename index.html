A Beginners Intro to Web Development

You will need a good text editor to get the most out of this workshop. Atom (atom.io) is recommended.

Objectives

By the end of this workshop you should be able to:

  • Name some different HTML Elements and Attributes
  • Create and view a simple webpage
  • Add some simple styling improvements to a webpage
  • Deploy a website to the internet!
  • Love Web Development

HTML: HyperText Markup Language

  • Language that web browsers use to render pages
  • First released in 1993
  • Designed to create web documents. Has since been extended for web applications

HTML TAGS

Used to define an element

<element>
<element></element>

SOME IMPORTANT HTML TAGS

<h1>Header Tag</h1>
<p>Paragraph Tag</p>
<img> Image Tag
<div>Divider Tag</div>
<a>Anchor Tag</a>

Reference:

https://developer.mozilla.org/docs/Web/HTML/Element

Attributes

Used to specify characteristics of an element

<tag attribute></tag>
<tag attribute="value"></tag>

Some important attributes

<div id="my-div"></div>
<div class="demo-div"></div>
<img src="/address/of/image.jpg">
<a href="https://benbru.com">benbru.com</a>

Cascading Style Sheets

  • Language for describing the presentation of a document
  • First released in 1996
  • Evolved alongside HTML
  • Most browsers support CSS 3

CSS rules


                                    element {
                                        rule: value;
                                    }
                            

                                    .class {
                                        rule: value;
                                    }
                            

                                    #id {
                                        rule: value;
                                    }
                            

                                    div {
                                        color: blue;
                                    }
                            

                                    .info-box {
                                        background-color: yellow;
                                        color: purple;
                                    }
                            

                                    #container {
                                        border: 1px solid black;
                                    }

Reference:

https://developer.mozilla.org/docs/Web/CSS
