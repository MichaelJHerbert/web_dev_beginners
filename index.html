A Beginners Intro to Web Development

Objectives

By the end of this workshop you should be able to:

  • Name some different HTML Elements and Attributes
  • Create and view a simple webpage
  • Add some simple styling improvements to a webpage
  • Love Web Development!

HTML: HyperText Markdown Language

  • Language that web browsers use to render pages
  • First released in 1993
  • Designed to create web documents. Has since been extended for web applications

HTML TAGS

Used to define an element

<element>
<element></element>

SOME IMPORTANT HTML TAGS

<h1>Header Tag</h1>
<p>Paragraph Tag</p>
<img> Image Tag
<div>Divider Tag</div>
<a>Anchor Tag</a>

Reference:

https://developer.mozilla.org/docs/Web/HTML/Element

Attributes

Used to specify characteristics of an element

<tag attribute></tag>
<tag attribute="value"></tag>

Some important attributes

<div id="my-div"></div>
<div class="demo-div"></div>
<img src="/address/of/image.jpg">
